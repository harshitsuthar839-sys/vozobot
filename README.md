# Vozo Bot

Vozo Bot is a custom bot for Slack that I built using Node.js and Slack Bolt. The Vozo Bot responds to commands. Is hosted on Hack Club Nest so the Vozo Bot can run all the time even when my computer is turned off.

## Features

* The Vozo Bot has a `/vozo-command. This checks how long it takes for the Vozo Bot to respond

* The Vozo Bot has a `/vozo-help` command. This shows you all the commands you can use with the Vozo Bot

* The Vozo Bot has a `/vozo-joke` command. This gives you a joke from the Vozo Bot

* The Vozo Bot has a `/vozo-command. This gives you a random fact about cats from the Vozo Bot

## Tech Stack

* The Vozo Bot uses Node.js

* The Vozo Bot uses Slack Bolt

* The Vozo Bot uses Socket Mode

* The Vozo Bot is hosted on Hack Club Nest

* The Vozo Bot code is on GitHub

## Installation

1. First you need to clone the repository, for the Vozo Bot

2. Then you need to install the dependencies for the Vozo Bot

```bash

npm install

```

3. Next you need to create a file called `.env` for the Vozo Bot

```env

SLACK_BOT_TOKEN=your_bot_token

SLACK_APP_TOKEN=your_app_token

```

4. Finally you can start the Vozo Bot

```bash

node index.js

```

## Deployment

The Vozo Bot is hosted on Hack Club Nest and it runs all the time so you can use the Vozo Bot whenever you want.
